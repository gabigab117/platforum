Pour les Forum Masters

L'espace d'administration

Un créateur de forum dispose d'un accès spécial à l'espace administration du forum.

Depuis la vue d'ensemble vous pouvez voir le nombre de membres, de sujets et de messages. Vous pouvez aussi modifier le logo de votre forum.

Depuis la gestion des membres, vous pouvez désactiver les comptes (mais pas les supprimer). Un membre désactivé ne peut plus participer au forum. Il est aussi possible d'accéder au profil d'un membre (vous pouvez aussi désactiver un compte depuis la page de profil).

Enfin, le constructeur. C'est depuis cette page que vous allez créer et organiser les catégories et sous-catégories.

Depuis le formulaire accessible en bas de page, vous pouvez créer une catégorie avec des sous catégories.
Index permet d'ordonner les catégories et sous-catégories. Donc une catégorie avec un Index de 1 suivra la catégorie avec un index 0. De même pour les sous-catégories.

Voici ce que cela donne une fois le formulaire (de la capture précédente) validé.

Autre

Dans une sous-catégorie le créateur du forum peut "épingler" les discussions pour qu'elles soient en tête de la liste des discussions. Pourquoi ne pas épingler une discussion "Annonces", "Règles" ou autre ? Vous pouvez réaliser l'opération inverse.

Le créateur du forum peut aussi gérer les messages de tous les membres (modifier, supprimer).

Depuis une discussion, vous pouvez modifier le sujet, la sous catégorie, ou même la supprimer.
